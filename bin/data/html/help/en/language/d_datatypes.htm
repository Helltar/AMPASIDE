The MIDletPascal Dialect \ Data types


The data types used in MIDletPascal can be divided into two categories: simple data types and complex data types.

Simple data types (also called primitive types) are:

  • boolean
  • char
  • integer
  • real
  • string
  • image
  • command
  • recordStore
  • http
  • resource
Complex data types are:
  • records
  • arrays
