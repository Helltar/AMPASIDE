Integer type


An integer type can represent any signed integer value between -2,147,483,647 and 2,147,483,647. The following operators can be used on integer types:

  • Comparison operators: =, <>, <, >, <=, >=
  • Addition operators: +, -
  • Multiplication operators: *, /, div, mod (there is no difference between / and div) (mod returns the remainder from the integer division)
  • Shift operators: shl, shr
