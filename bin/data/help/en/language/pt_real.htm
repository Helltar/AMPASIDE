Real type


The real type is used to represent real numbers (decimal numbers). The mobile devices do not have hardware support for real numbers, so MIDletPascal uses software emulation instead. In the Fixed-point mode the precision is lower, so expect aproximated values. For more precision enable the Floating-point mode.

Also, be sure to remember that the software emulation is slow, so you should be aware that extensive mathematical applications will run very slow on mobile devices. Use of real numbers in a MIDlet will also increase the size of MIDlet a few KBs.

The following operators can be used on real types:

  • Comparison operators: =, <>, <, >, <=, >=
  • Addition operators: +, -
  • Multiplication operators: *, /

MIDletPascal also provides about 20 functions for working with real numbers and a predefined constant 'pi'.
